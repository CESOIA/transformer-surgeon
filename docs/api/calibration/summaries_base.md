# CalibrationSummary (ABC)

Abstract contract for a statistic computed over calibration batches.
Instances are singletons stored in `SUMMARY_REGISTRY`. Per-scheme
accumulation state lives in `SummaryRuntime`.

See [Concepts](../../concepts.md) for the full explanation.
